Marcador de Posición para Imagen

Este archivo es un marcador de posición para una imagen o GIF que se utilizará en la aplicación Poker Planning Pro.

Para reemplazar este marcador de posición, cree una imagen o GIF con el mismo nombre que este archivo (pero con la extensión .gif o .png) y colóquelo en este directorio.

Nombres de archivos necesarios:
  • planning-estimation.gif
  • planning-series.gif
  • planning-roles.gif
  • planning-demo.gif

Las imágenes deben mostrar las siguientes funcionalidades:

  • planning-estimation.gif: Demostración de estimación en tiempo real
  • planning-series.gif: Demostración de diferentes series de estimación (Fibonacci, T-shirt, etc.)
  • planning-roles.gif: Demostración de roles y permisos
  • planning-demo.gif: Demostración completa del flujo de una sesión de Planning Poker

Nota: Para obtener mejores resultados, las imágenes deben tener una relación de aspecto de 16:9 y un tamaño recomendado de 800x450 píxeles o mayor.
